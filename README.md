# Personal Blog Website

## Description

A React application demonstrating the use of Components and Props.

## Features

* Header Component
* About Component
* Article List Component
* Article Component
* Dynamic rendering using props

## Run Locally

### Install Dependencies

npm install

### Start Development Server

npm run dev

### Run Tests

npm test
